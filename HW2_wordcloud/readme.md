# 簡介
運用hash技巧並搭配python內建或外部函式建立圖形化的文字雲。
**限制：僅用於英文文本分析**

# AI技術運用
- 使用AI完善UI介面
- 請AI協助增列停用字，並在在主運算code裡增設縮寫字詞的刪減
